Jika Kita mengklik tombol "Kirim", maka data formulir akan dikirim ke halaman bernama "/action_page.php"

1. Untuk membuat tombol kirim dan reset cukup diperlukan input type submit dan reset. Setelah menuliskan nama lengkap lainnya ketika menekan tombol reset maka nama pada input awal akan kembali seperti semula.
2. Atribut method menentukan cara mengirim data formulir. Untuk method get digunakan untuk mengirim formulir dengan variabel URL, sedangkan method post digunakan untuk mengirim formulir sebagai transaksi kiriman HTTP.
